### 01-01: hash generator
Server returns hash from current timestamp

Test link: <http://5.19.210.159:8080>

Accepted values:
* hash = md5 | sha1
* type = plain | json

Example request with cURL: curl -sS 'http://5.19.210.159:8080?hash=md5&type=json'
